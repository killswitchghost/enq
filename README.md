# Enqbator Website Templates v1.1

Front-end code for the Enqbator Website Templates.
<br><br>

https://enqbator-app.vercel.app/
<br><br>

## Github Deployment

## How to use

```sh
cd /
npm install
npm run build
npm run dev
```
